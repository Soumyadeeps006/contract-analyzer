## Overview
The service allows you to:
- Upload contract PDFs/DOCX files.
- Use a locally hosted LLM (Ollama) to extract key clauses.
- Evaluate clauses against configurable compliance rules (GDPR, SOC 2, custom).
- Store documents and analysis results in PostgreSQL.
- Perform vector‑based similarity search with `pgvector`.
- Serve a React UI for uploading files, reviewing analysis
